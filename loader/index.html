Setup

To use Bulldog Book, you need to fetch some info from Yale's directory. The extension will open Yale Facebook and guide you through the process. This won't work if you access Yale Facebook by yourself. You have to do it by clicking on the link below.
Read the NOTICE section below and then click the blue button to start.

What it looks like on someone's Facebook profile:

NOTICE

You will need to have access to Yale Face Book in order to properly install this extension.

Bulldog Book is a tool that keeps track of students names, class years, residential colleges and suitemates' names, using data downloaded directly from Yale Face Book to your computer. All student data is kept in your computer; not in any outside servers. The authors of this extension exclude any liablity for misuse of this data by users of this extension or third-parties. Users of this extension are strongly discouraged from installing Bulldog Book in a public computer. Moreover, all users of this extension are expected to comply with the Yale Face Book terms of use, which can be found in their website.

By clicking on the "Go To Yale Facebook" button above, you indicate that you have read and agreed with the terms above.
