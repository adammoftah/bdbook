Notice

You will need to have access to Yale Face Book in order to properly install this extension.

Bulldog Book is a tool that keeps track of students names, residential colleges and suitemates' names, using data downloaded directly from Yale Face Book to your computer. All student data is kept in your computer; not in any outside servers. The authors of this extension exclude any liablity for misuse of this data by users of this extension or third-parties. Users of this extension are strongly discouraged from installing Bulldog Book in a public computer. Moreover, all users of this extension are expected to comply with the Yale Face Book terms of use, which can be read in their website.

By clicking on the "Go To Yale Facebook" button above, you indicate that you have read and agreed with the terms above.

Not a Mark Zuckerberg production.
